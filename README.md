## Description

[Nest](https://github.com/nestjs/nest) framework based some hello world app with some CRUD User func
for start you need postgres, or you can run command 

```bash
$ docker-compose up
```

and add .env file (you can only rename example one)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
